📅 定时任务配置建议

现在推荐优先在管理后台的“智能设置”里配置自动更新时间。下面的 cron 命令可作为备用方案：

# 每天凌晨 2:00 自动同步
0 2 * * * cd /path/to/project && python auto_sync.py >> logs/sync.log 2>&1

或者每周一凌晨 3:00：

# 每周一凌晨 3:00 自动同步
0 3 * * 1 cd /path/to/project && python auto_sync.py >> logs/sync.log 2>&1
